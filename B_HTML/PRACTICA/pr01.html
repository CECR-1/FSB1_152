    Iframes

    Un iframe es un elemento HTML que permite incrustar otro documento HTML dentro del documento actual. Es útil para mostrar contenido externo, como videos, mapas o páginas web.

    Formularios

    Un formulario en HTML es un elemento que permite a los usuarios ingresar y enviar datos a un servidor web mediante campos como cajas de texto, botones, casillas y listas.

    Semántica

    La semántica en HTML se refiere al uso de etiquetas que describen el significado y la función del contenido, mejorando la estructura, accesibilidad y comprensión tanto para navegadores como para personas.

    Elemento Descripción
    <section> Para crear secciones de una web.
    <nav> Para especificar un menú de navegación.
    <header> Para definir la cabecera.
    <footer> Para definir el pie de página.
    <main> Para definir el contenido principal.
    <article> Para indicar secciones de contenido.
    <aside> Para definir una barra lateral.
    <figure> Para representar contenido gráfico.
    <figcaption> Para describir el contenido de <figure>.
    <hgroup> Para agrupar titulares consecutivos.
